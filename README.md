# Smart Environmental Monitor

## Objective
Monitor temperature and humidity using a DHT22 sensor connected to an Arduino Uno.

## Components
- Arduino Uno
- DHT22 Sensor
- Jumper Wires

## Connections
- VCC → 5V
- DATA → D2
- GND → GND

## Features
- Reads temperature
- Reads humidity
- Updates every 2 seconds
- Displays data on Serial Monitor

## Output Example
Temperature: 24.00 C
Humidity: 40.00 %

## Simulation
Built and tested using Wokwi.
